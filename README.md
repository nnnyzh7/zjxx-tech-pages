# 信息科技教学网页集合

这是用于信息科技课堂的互动网页集合。

在线访问：

- https://nnnyzh7.github.io/zjxx-tech-pages/

## 网页入口

### 拼豆编码工坊

- `拼豆编码工坊.html`
- 二进制解码挑战、MARD 标准拼豆色号、内置图案、上传图片生成拼豆图纸。
- `assets/bead-studio/` 是这个页面运行需要的数据文件夹，不是学生需要点击的入口。

### 四下第 2 课：图像编码

- `四下第2课_图像编码.html`
- `图片.jpg`：该页面使用的配套图片素材。

### 四下第 3 课：字符编码

- `四下第3课_字符编码.html`
- `解密文档.txt`
- `雷锋日记.png`

### 四下第 4 课：声音编码

- `四下第4课_声音编码.html`

### 四下第 7 课：数据校验

- `身份证校验.html`
- `奇偶校验.html`

### 四下第 9 课：人机编码有差异

- `四下第9课_人机编码有差异.html`

## 说明

`index.html` 是 GitHub Pages 的默认首页，用来整理所有教学网页入口。学生打开仓库主页时会先看到集合导航，再进入对应课堂活动。
